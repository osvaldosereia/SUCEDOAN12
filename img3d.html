Limitações desta versão

Esta é uma versão HD em HTML único.
• Aceita imagem e SVG, processando tudo no navegador.
• Resolução interna até 2048 px para preservar detalhes pequenos.
• STL separado por cor é exportado em binário para reduzir tamanho.
• Quanto maior a resolução, mais pesado fica o navegador e o arquivo final.
• O modo principal desta versão é empilhado por cor, ideal para extrusor único com troca manual de filamento.
• Também exporta STLs separados caso você queira montar ou testar as peças individualmente.
