Como este sistema salva

Ele puxa a base somente do Firebase. Quando você altera um produto, o botão verde envia um PATCH somente para aquele produto e somente com estes campos: estoque, validade, embalagem, gondola e prateleira.

Nenhum outro produto é enviado. Nenhum arquivo do GitHub é alterado por esta tela.
