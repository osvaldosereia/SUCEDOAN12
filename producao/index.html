Produtos

Em cada card aparece automaticamente em quais cestas o produto está. Clique em “Editar cestas” para marcar ou desmarcar.

Administração das cestas

Selecione uma cesta, veja a lista de produtos, substitua itens, altere quantidades, adicione, remova ou crie uma nova cesta.

JSON das cestas

Área para conferir, copiar, importar ou exportar o JSON final das cestas básicas.

Atenção: a fonte da verdade das cestas é este JSON. Os cards dos produtos apenas leem o JSON para mostrar as marcações.
